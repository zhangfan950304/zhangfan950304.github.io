react中创建组件的两种方式

第一种：使用function构造函数

-使用构造函数创建组件，把构造函数的名称，当作组件的名称，以html标签的形式映入到页面中即可
组件的首字母必须要大写

  • 基本使用
  • 构造函数创建的组件中传值
  • 构造函数创建的组件中传递对象的全部内容

使用剩余函数传递
构造函数的参数部分需要定义props，才能使用
通过props得到的任何数据都属于只读形式，不能重新赋值

第二种：使用class继承React.Component来实现

  • 基本使用方式

此种方式创建的组件可以定义自己的私有数据
此种方式创建的组件需要写constructor({}),f方法体中需要调用super()
this.state中的数据是可读可写的

-该方式创建的组件中传值
在constructor中定义了props可直接使用props.xx的形式获取参数值
该方式创建的组件中可直接使用this.props.xx的形式获取参数值

react的生命周期

  • react的生命周期分三个阶段
  • 挂载 :(componentWillMount() componentDidMount()）
    1. 先执行construcor及this.state里的初始化数据
    2. componentWillMount()组件将要开始挂载
    3. 组件挂载render解析渲染
    4. componentDidMount()组件挂载完成
  • 渲染 :(componentWillReceiveProps(nextProps,nextState)shouldComponentUpdate(nextProps,nextState) componentWillUpdate()componentDidUpdate())
    1. shouldComponentUpdate()确定是否要更新数据
    2. 当上函数返回true时进行更新
  • 卸载 :(componentWillUnmount())
